Labels in forms

All of the form elements below have visible labels. Some of these visible labels are not logically associated with the form element. Use the tab key to navigate to each form element. Some text will appear by the element with what a screen reader user will hear. If the visible label is not associated with the form element, all you will see is the prompt, not the name of the element.

A quick way to verify that the visual label is associated with a form element is to click on the label. If the form element receives focus, the label and the element are logically associated. Give it a try!
