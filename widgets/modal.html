Guidelines

  • Add role="dialog" to the first node of the modal markup
  • Add aria-hidden="true" to the first node of the modal markup
  • Add role="document" to the second node of the modal markup
  • Add aria-describedby="id_of_the_header_in_the_modal"
  • Regardless of framework, focus passes to the modal when invoked, focus is trapped in modal when displayed, focus passes back to element that invoked it when dismissed.
