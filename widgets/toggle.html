Guidelines


  • Keyboard
    • The switch receives and displays keyboard focus state clearly
    • The visible focus treatment is independent of the state of the switch
    • The focus and state flags of the switch do not rely on color only
    • The switch responds to Space or Enter keys to change state
  • Name, role, value
    • The toggle switch provides its name, initial state and role to the accessibility tree when encountered
    • The toggle switch changes the state in the accessibility tree when toggled via aria-pressed="true/false" when a button
