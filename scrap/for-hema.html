Some tools to help test

Since we cannot test directly, here are some simple tests that Hema can perform herself. These come from our page on non-technical tests

  1. Tab through screen
  2. Try to operate all the controls with the keyboard
  3. Use the filters with a keyboard only
  4. Greyscale screen, drag this link to toolbar: Grayscale
  5. Zoom, (Windows: Control +) till you reach 200%
  6. Bump text size
  7. Change text spacing, drag this to toolbar Change text spacing
