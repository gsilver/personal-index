Table - visual demo

Visually, we can determine the x and y of every cell by triangulating the column and row labels with the cell.

Hover over cells to see an exagerated version of this.

In order for screen readers to be able to do the same triangulation the table needs to be properly configured, on the web or in documents like MS Word.
